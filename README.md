# WellAnalyzer
- Реализована асинхронная загрузка данных по скважинам из CSV
- Добавлена валидация строк и интервалов
- Реализован расчет сводной информации по скважинам
- Добавлен экспорт сводной информации в JSON
- Реализован UI
- Добавлены xUnit-тесты для основной бизнес-логики

# Инструкция по запуску
- Откройте терминал в корневой папке проекта, где находится файл решения `WellAnalyzer.slnx`
- Восстановите зависимости: `dotnet restore`
- Соберите решение: `dotnet build`
- Запустите приложение: `dotnet run --project .\WellAnalyzer\`

# Реализованные дополнительные пункты:
1. Асинхронная загрузка файла:
- Загрузка CSV-файла реализована асинхронно с использованием async/await, чтобы не блокировать UI во время чтения и обработки данных
  
2. Экспорт сводной информации в JSON:
- Добавлена возможность сохранить рассчитанную сводную информацию по скважинам в JSON-файл
  
3. Unit-тесты:
- Добавлены xUnit-тесты для основной бизнес-логики:
  - Проверка расчета сводной информации по скважине
  - Проверка валидации данных и обработки ошибок
